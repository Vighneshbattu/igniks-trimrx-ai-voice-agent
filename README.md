# Igniks – TrimRX AI Voice Agent (Jessica)

AI voice agent built on VAPI for medication refill check-ins.

## Tech Stack
- **Platform:** VAPI
- **LLM:** GPT-4.1 (OpenAI)
- **STT:** Deepgram (100ms)
- **TTS:** ElevenLabs Turbo v2.5 (400ms)
- **Calling:** Twilio
- **Automation:** n8n (outbound trigger)
- **Total Latency:** ~1300ms

## Built for
CareCaller Hackathon 2026 — Problem Statement 2: AI Voice Agent Simulator
